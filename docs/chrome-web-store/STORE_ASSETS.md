## Store listing description (1000 chars max)

NOTEtoolsLM is the open-source command center for Google NotebookLM power users.

Manage notebooks from a fleet dashboard, store artifacts in a local vault, and launch eight content prefabs (podcast, briefing, explainer, deck, mind map, and more) from the Chrome side panel.

Features:
- Fleet dashboard with real-time WebSocket sync
- Chrome MV3 side panel on notebooklm.google.com
- Local vault for artifact metadata and downloads
- SDK-backed generation with graceful fallback
- Portfolio export to cineforge and lookBOOK formats
- 100% local-first — your vault stays on your machine

Requires a local NOTEtoolsLM server (`npm start`) and optional NotebookLM SDK auth (`npx notebooklm-sdk login`).

Privacy-first: no cloud telemetry. Data stored in local SQLite and chrome.storage.local.
