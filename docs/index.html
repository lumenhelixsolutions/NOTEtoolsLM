Built for power users

NOTEtoolsLM complements NotebookLM with local tooling the official UI does not provide.

📚

Sources export

Extract all sources from the current notebook, browse them in the side panel, and download individual .md files or a full ZIP — stored locally in your browser.

🎬

Studio vault

Auto-detect audio, video, slides, mind maps, and reports. Store artifacts in a typed local vault with one-click download and bulk operations.

🚀

Prefab generation

Eight production templates — podcasts, briefings, explainers, decks, tutorials — injected directly into NotebookLM from a floating toolbar.

🛰️

Fleet dashboard

Optional local server on localhost:3000 for notebook fleet sync, SDK orchestration, webhooks, and real-time pipeline progress.

How it works

Two components, one workflow. Use the extension alone for sources and vault, or add the server for fleet-scale automation.

1

Open NotebookLM

Navigate to any notebook at notebooklm.google.com.

2

Open NOTEtoolsLM

Click the extension icon → side panel with Artifacts and Sources tabs.

3

Extract, store, export

Sync studio media, extract sources to Markdown, or launch prefabs — all user-triggered, all local.

🔒 Data stays on your machine 📥 Sources → Markdown + ZIP ⚡ SDK when you need fleet scale 🧩 MV3 Chrome extension
